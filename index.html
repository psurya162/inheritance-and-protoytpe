Q1 : Create 2 objects - parent and child. Attach methods to parent and use those methods in child object using parents prototype

Ans:

        
const parent = {
  name: "John",
  greet: function() {
    console.log(`Hello, my name is ${this.name}`);
  }
};


const child = Object.create(parent);
child.name = "Alice";


child.greet(); 
// Output: Hello, my name is Alice
In this code, we create a parent object with a name property and a greet method. Then, we create a child object that inherits from the parent object and override the name property in the child. Finally, we use the greet method from the parent's prototype in the child object, resulting in the output: "Hello, my name is Alice."

Q2 : Write code to explain prototype chaining

Ans: 

         
          function Person(name, age) {
            this.name = name;
            this.age = age;
          }
          
          
          Person.prototype.sayHello = function() {
            console.log(`Hello, my name is ${this.name}, and I'm ${this.age} years old.`);
          };
          
         
          const person1 = new Person("Alice", 25);
          
          
          function Student(name, age, studentId) {
            Person.call(this, name, age); // Call the parent constructor.
            this.studentId = studentId;
          }
          
          
          Student.prototype = Object.create(Person.prototype);
          Student.prototype.constructor = Student;
          
         
          Student.prototype.study = function(subject) {
            console.log(`${this.name} with student ID ${this.studentId} is studying ${subject}.`);
          };
          
          
          const student1 = new Student("Bob", 20, "S12345");
          
         
          person1.sayHello();  
          student1.sayHello(); 
          student1.study("Math");
This code prototype chaining by creating two constructor functions, "Person" and "Student." "Student" inherits properties and methods from "Person" through the prototype chain. The code creates instances of both "Person" and "Student" and showcases how properties and methods are accessed through prototype chaining for each object.

Q3 : Add a method to calculate sum of all elements in Array in array's protype, use that method to calculate sum for multiple arrays

Ans : 

     
Array.prototype.Sum = function() {
  return this.reduce((accumulator, currentValue) => accumulator + currentValue, 0);
};


const arr1 = [1, 2, 3, 4, 5];
const arr2 = [10, 20, 30];
const arr3 = [2, 4, 6, 8];


const sum1 = arr1.Sum();
const sum2 = arr2.Sum();
const sum3 = arr3.Sum();


console.log("Sum of arr1:", sum1);
console.log("Sum of arr2:", sum2);
console.log("Sum of arr3:", sum3);

Q4 : Write a JavaScript function to retrieve all the names of object's own and inherited properties.

Ans : 

      function getAllPropertyNames(obj) {
        const propNames = [];
      
        for (let key in obj) {
          if (obj.hasOwnProperty(key)) {
            propNames.push(key);
          }
        }
      
        return propNames;
      }
      
      // Example usage:
      const myObject = {
        a: 1,
        b: 2,
      };
      
      const inheritedObject = Object.create(myObject);
      inheritedObject.c = 3;
      
      const allPropertyNames = getAllPropertyNames(inheritedObject);
      
      console.log(allPropertyNames);
In this code The getAllPropertyNames function retrieves and returns all property names (both own and inherited) of an object by iterating through its properties using a for...in loop and checking each property's ownership using hasOwnProperty.
